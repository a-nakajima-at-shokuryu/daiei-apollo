```graphql

  tableList {
    id 
    name
    title 
    filename 
  }
}
```

**テーブル定義取得（売掛トラン）**
```graphql
query {
  describe(tablename:"urikaktrn") {
    info {
      id 
      name 
    }
    columns {
      no
      title 
      prefix 
      name
      type
      size 
      iskey 
      biko 
    } 
  }
}
```
